# CSE 326: Microprocessor and Microcontroller

Microprocessors **(MPU)** are high-performance CPUs, such as Intel Core or AMD Ryzen, requiring external memory/peripherals for general-purpose computing.

Microcontrollers **(MCU)**, such as ARM Cortex-M or AVR, integrate CPU, RAM, ROM, and I/O on a single chip, designed for specialized, low-power, cost-effective embedded tasks.
